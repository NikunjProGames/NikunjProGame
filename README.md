# Nikunj Pro Games

A Cool Trendy FREE Online Gaming Website

## What it is

A fully featured browser gaming hub with animated UI, a working mini-game, leaderboard, trending section, login modal, and game-launch overlay — no build tools or backend required.

## Key technologies

- **Google Fonts** — Orbitron & Rajdhani for the futuristic look
- **Font Awesome 6** (CDN) — icons throughout the UI
- **Vanilla JavaScript** — particle system, mini-game engine, filter tabs, modals
- **CSS custom properties + animations** — neon glow effects, glassmorphism, keyframe animations
- **HTML5 Canvas** — particle background and the in-browser click-the-target mini-game
